# 마이크로니들 연속혈당측정기(CGM) 사업화 마일스톤

> 대상 제품: 마이크로니들 기반 최소침습 연속혈당측정 시스템
> (센서 패치 + AD5941 전기화학 계측 + BLE + 모바일 앱)
> 작성 기준일: 2026-07-15 · 담당: kjh5001212

---

## 0. 제품 개요 및 현재 위치

**제품 컨셉**: 피부 각질층만 통과하는 미세바늘(microneedle) 어레이로 진피 상부의
**조직간질액(ISF)** 에 접촉, 효소(GOx/GDH) 전극에서 발생하는 나노암페어급 전류를
측정해 혈당을 추정하는 최소침습형 연속혈당측정기(CGM).

**기술 스택 (본 저장소가 담당하는 부분)**
- 계측 엔진: nRF52832 + AD5941 3전극 정전위기, +0.5V 고정 바이어스, nA급 크로노암페로메트리
- 통신: BLE(Nordic UART Service), 무손실 오프라인 버퍼링, 재접속 재전송
- 앱: 실시간 전류-시간 그래프, CSV 기록/내보내기

**현재 성숙도 (TRL 4 근방)**
- ✅ 계측 회로/펌웨어 구현, AD5941 하드웨어 검증(ADIID 0x4144)
- ✅ 안드로이드 앱 BLE 스트리밍/기록 파이프라인
- 🔲 마이크로니들 센서(효소 전극) 미개발 — **사업화 최대 리스크**
- 🔲 더미셀→실제 생체 신호 검증, 혈당-전류 캘리브레이션 미완
- 🔲 인허가·임상·양산 체계 없음

> **핵심 메시지**: 전자/계측은 앞서 있으나, 사업 가치는 **마이크로니들 효소센서의
> 정확도·수명·생체적합성**에서 결정된다. 마일스톤은 이 병목에 자원을 집중한다.

## 2. 단계별 상세 마일스톤

### M1 — 마이크로니들 센서 개념검증 (PoC) · +0~6개월
사업의 사활이 걸린 센서를 최우선으로 검증한다.

- [ ] **효소 화학 선정**: GOx vs GDH-FAD (산소 의존성/간섭 비교), 매개체(mediator) 선택
- [ ] **전극 구조**: WE/RE/CE 3전극 미세바늘 어레이 시제작 (금속/전도성 폴리머, 도금 Pt/Ag-AgCl)
- [ ] **고정화**: 효소 고정막 + 확산제한막(glucose-limiting membrane) 코팅
- [ ] **in-vitro 검증**: PBS/인공 ISF에서 2~30 mM 글루코스 농도별 nA 응답 곡선, 선형성/감도
- [ ] **계측 연동**: 본 저장소 펌웨어로 실제 센서 전류 획득(더미셀→실센서 전환)
- [ ] 특허 선행조사 및 **가출원 1건**

**Exit**: 생리 농도 범위에서 재현 가능한 농도-전류 상관(R²>0.98), 감도 규격 정의
**리스크**: 효소 안정성/신호 드리프트 → 초기부터 수명·드리프트 데이터 축적

---

### M2 — 착용형 통합 프로토타입 · +6~12개월
- [ ] 패치 폼팩터(어플리케이터, 접착, 방수 IP27급) 설계
- [ ] 전자부 소형화: nRF52832+AD5941 → 유연 PCB / 저전력 최적화(코인셀 7~14일 목표)
- [ ] 온도보정, 자동 캘리브레이션(팩토리 캘 or 1점 캘) 알고리즘 초안
- [ ] in-vitro 정확도·수명(7일 연속) 시험, 드리프트 보정 로직
- [ ] 앱: 혈당값(mg/dL) 환산, 추세 화살표, 경보(저/고혈당) UI
- [ ] **사이버보안/데이터**: BLE 페어링 보안, 개인정보(건강정보) 처리방침 초안

**Exit**: 7일 착용형 프로토타입에서 in-vitro MARD 목표(<15%) 근접, 배터리·방수 충족

---

### M3 — 설계 동결·설계검증(DV)·전임상 · +12~20개월
규제 진입을 위한 품질·안전 기반 구축.

- [ ] **ISO 13485 품질경영시스템(QMS)** 구축 착수, 설계이력파일(DHF) 관리
- [ ] **위험관리 ISO 14971** (위해요인 분석, FMEA)
- [ ] **생체적합성 ISO 10993** (세포독성·감작성·자극성·피내반응 — 피부접촉/침습 등급)
- [ ] **전기안전/EMC IEC 60601-1 / -1-2**, 무선 IEC 60601-1-11(홈헬스케어)
- [ ] **소프트웨어 IEC 62304** 생명주기, 사용적합성 IEC 62366
- [ ] 설계검증(DV): 정확도/수명/방수/낙하/멸균(해당 시) 시험
- [ ] 동물시험(필요 시) 및 초기 인체 파일럿(탐색 임상) 계획 수립
- [ ] **설계 동결(Design Freeze)** — 이후 변경은 형상관리로 통제

**Exit**: DV 리포트 통과, 생체적합성/전기안전 시험 성적서, 임상 프로토콜 IRB 준비 완료

---

### M4 — 인허가용 확증 임상 · +20~32개월
- [ ] 임상시험계획(식약처 임상 승인 또는 IDE 등 관할별) 승인
- [ ] **확증 임상**: 정맥/모세혈 기준치 대비 정확도, 목표 **MARD ≤ 12%**
      (프리미엄 목표 iCGM/특수관리 기준 근접), Clarke/Consensus Error Grid A+B ≥ 99%
- [ ] 다양한 조건(운동/식후/저혈당/피부부위/체온) 하위군 분석
- [ ] 삽입 통증/피부 이상반응 등 안전성 평가
- [ ] 임상통계 분석계획(SAP) 및 최종 임상시험결과보고서(CSR)

**Exit**: 사전 정의 정확도 목표 통계적 충족, 중대 이상반응 없음, CSR 확정

---

### M5 — 인허가 취득 · +32~40개월
- [ ] **국내(식약처/MFDS)**: 의료기기 품목 분류 확정(연속혈당측정시스템, 통상 3등급),
      GMP 심사 + 기술문서 심사 → **품목허가**
- [ ] **미국(FDA)**: iCGM 특수관리(De Novo 확립 후 510(k)) 경로 검토, QSR/QMSR 대응
- [ ] **유럽(CE)**: MDR (EU) 2017/745, Notified Body 심사, 임상평가보고서(CER)
- [ ] 라벨링/IFU/UDI, 시판후조사(PMS)·이상사례 보고 체계

**Exit**: 최소 1개 관할(우선 국내) **허가 취득**, 판매 가능 상태

---

### M6 — GMP 양산 이관·출시 · +40~48개월
- [ ] **제조 이관(NPI)**: 센서(효소코팅·멸균) 자동화 라인, 전자부 EMS 파트너
- [ ] 공정검증 IQ/OQ/PQ, 수율·양산 원가(BOM) 목표 확정
- [ ] 멸균 밸리데이션(해당 시), 안정성/유효기간 시험(가속·실시간)
- [ ] 초도 물량 생산, 판매/유통·CS·리버스 물류 체계
- [ ] **초도 출시(제한 출시 → 정식 출시)**, 초기 사용자 코호트 리얼월드 데이터

**Exit**: GMP 양산 안정화, 초도 판매 개시, 필드 성능 모니터링 가동

---

### M7 — 시장 확대·수가·글로벌 · +48개월~
- [ ] **건강보험 급여/요양급여 등재** 또는 비급여 시장 전략, 보험사·병원 채널
- [ ] 해외 인허가 순차 확대(미국·유럽·동남아 등), 현지 파트너/디스트리뷰터
- [ ] 차기 버전: 인슐린 펌프 연동(자동인공췌장), 멀티애널라이트(케톤/젖산), 저원가 판형
- [ ] 처방/약국/DTC 등 채널 다변화, 구독형 소모품 매출 모델

**Exit**: 지속 성장 매출, 재구매(소모품) 궤도, 다국가 사업 기반
